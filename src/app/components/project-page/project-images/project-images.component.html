Expense Planner
Track and manage your expenses on the go with our user-friendly app.
The Challenge
One of the core challenges was creating an intuitive and visually appealing user interface that would simplify expense categorization and budget planning.
Scope, Contribution & Design Rationale
As a key contributor, I was responsible for developing the entire frontend using Flutter and implementing the backend APIs, ensuring a smooth and responsive user experience.
The design was kept minimalist to ensure that users can quickly and easily access and understand their financial information, allowing them to make informed decisions.
